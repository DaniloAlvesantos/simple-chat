# Simple Chat Website

This is a simple chat application built with **Vite**, **ShadCN**, and **Firebase**. The app allows users to send and receive messages in real-time.

## Features

- Real-time messaging
- User authentication with Firebase
- Simple UI using ShadCN components
- Mobile-responsive design

## Technologies Used

- **Vite**: Fast development environment and build tool
- **ShadCN**: A UI component library for building modern web interfaces
- **Firebase**: Backend-as-a-Service for authentication and real-time database
